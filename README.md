#ğŸ“ŒProject Overview
This project is a Python-based desktop application that provides real-time weather information through a simple, modern, and intuitive graphical user interface. The application is built using CustomTkinter, giving it a clean and minimal design while maintaining ease of use.
The app integrates with the OpenWeather API to fetch live weather data for different cities, including temperature, humidity, and current weather conditions. It demonstrates practical usage of API integration, HTTP requests, and environment variable management for secure handling of API keys.

This project follows a modular code structure, separating UI components, configuration, and logic into multiple files, which improves readability and maintainability. It is ideal for beginners learning Python GUI development as well as for academic projects and portfolio use.
